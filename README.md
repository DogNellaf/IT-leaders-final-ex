# IT-leaders-final-ex

Проект для обучения и оценки нейросети на датасете CIFAR-10.

Модель обучается на данных из папки `cifar-10-batches-py`, после чего сохраняется в файл `model.keras`.

## Что делает проект

- Загружает тренировочные данные из файлов `data_batch_1` ... `data_batch_5`
- Загружает тестовые данные из файла `test_batch`
- Нормализует признаки (делит значения пикселей на 255)
- Обучает полносвязную нейросеть для классификации 10 классов
- Оценивает качество на тестовой выборке
- Сохраняет обученную модель в `model.keras`

## Требования

- Python 3.9+

  - scikit-learn

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python answer.py
```

Во время запуска:

- если `model.keras` существует, она будет загружена;
- если файла модели нет или загрузка не удалась, создается новая модель;
- выполняется обучение (`EPOCH_COUNT = 100`), затем выводится `Test accuracy`.

## Архитектура модели

Текущая архитектура из `answer.py`:

- Dense(64, relu)
- Dense(256, relu)
- Dense(64, relu)
- Dense(10, softmax)

Функция потерь: `sparse_categorical_crossentropy`  
Оптимизатор: `adam`  
Метрика: `accuracy`

## Полезные замечания

- Входные данные в скрипте подаются как вектор признаков длины 3072 (32×32×3, развернутые в строку).
- Для воспроизводимости и контроля обучения можно добавить фиксированные seed и callbacks (например, EarlyStopping).

## Возможные улучшения

- Перейти на сверточную архитектуру (CNN), что обычно дает лучшую точность на CIFAR-10.
- Добавить валидационную выборку и раннюю остановку.
- Вынести гиперпараметры (epochs, batch size, learning rate) в аргументы командной строки.
